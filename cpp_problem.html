問題: フィボナッチ数列の計算

フィボナッチ数列のn番目の数字を計算するプログラムを作成します。

解法:

次のC++コードは、再帰的にフィボナッチ数を計算します。

#include using namespace std; int fibonacci(int n) { if (n <= 1) { return n; } return fibonacci(n - 1) + fibonacci(n - 2); } int main() { int n; cout << "n番目のフィボナッチ数を計算します: "; cin >> n; cout << "フィボナッチ数列の " << n << " 番目の数: " << fibonacci(n) << endl; return 0; }

このプログラムは再帰を使用してフィボナッチ数を計算します。効率を改善するためには、動的計画法を使用することをお勧めします。
